心理学浓汤宝（六）学习

发表于 2017-07-08 更新于 2021-12-06 分类于 psychology

如果请你对“学习”下一个定义，会是什么呢，是类似“啃书本”的知识概念的记忆和理解，或是像“学自行车”这样的身体协调运动能力的训练。如果从心理学的角度看，学习的范畴会大得多，即行为或心理过程被持久改变的过程，都可称之为“学习”。

  某天，你在家里被椅子撞了一下，以后你经过那里时就会留意椅子的摆放位置，这便是“学习”，严格来说属于“操作性条件作用”。实际上，除去本能的简单反射和数量有限的先天行为，所有人类的能力，都来自于学习。我们可以把学习大致分成几种形式，按照从简单到复杂的顺序，依次是：

  •   经典条件作用

  •   操作性条件作用

  •   认知学习

经典条件作用

这个理论来自著名的“巴普洛夫和他的狗”的实验，其实验过程通过下图可以说明。

  通过使食物刺激和一个无关的声音刺激配对出现，反复几次后，生物便会在单独的声音刺激下出现与食物刺激相同的反应。通过这种学习，生物体能够学会预示食物，危险，交配机会等的线索，这能促进生物体的生存。

  这种习得的关联，并不会永久存在，当两种刺激不再配对出现时，生物体会逐渐“学会”不再对无关的刺激作出反应，实际上这也是一种学习，心理学上称之为“消退”。

  经典条件作用的一个经典运用是帮助治疗化疗反应。大多数患者会在化疗后感到恶心，而化疗常常是在进食之后进行的，因此患者会将正常的食物与恶心关联起来，从而厌恶进食以至于营养不良。除了避免在进食后进行化疗，医生还会在化疗前给病人一种“替罪羊”刺激，比如一种口味特殊的糖果，患者只会对这种不寻常的味道产生厌恶，而不会对正常食物产生反应。对于一些病人而言，这种方法可能意味着生死之别。

操作性条件作用

  你可以运用经典条件作用教会狗听到声音就分泌唾液，但你不能用这种方法教它们坐起来或翻滚。这是为什么呢，前者是一种被动反应，而后者是自主的反应。不过这些“自主的”反应实际上被奖惩所控制。这次的实验是桑代克和他的猫。

  桑代克的猫面临一个问题：如何打开迷箱的门以获得外面的食物奖励。为了解决这个问题，动物们要不断地尝试各种行为。起初，它们的行为看似随机，但渐渐地，它们消除了那些无效的行为。当它们的行为获得良好的效果时，动物就会重复同样的策略，或者说，它们学会了如何打开迷箱。

  这里所说的奖励，心理学上称之为“强化物”。对大多数人而言，食物，金钱和性都能发挥强化物的作用，关心，赞扬和微笑也能够起到类似作用。

  除了获得强化物来强化行为，消除一种不良的刺激也有相同的效果。比如，使用雨伞来避免淋湿，系上安全带来消除恼人的警告声，这些称之为“负强化”。

  与强化对应的是惩罚，后者同样也能影响行为，但它通过施加令人厌恶的刺激或者剥夺某种行为的权利（强化物）来实施。惩罚往往会带来一些副作用，可能会引起被惩罚者的逃避或攻击性行为，或者会让学习者变得恐惧和忧虑，干扰了其学习能力。

  和经典条件作用的“消退”一样，强化和惩罚都需要不断的巩固，否则效果就会下降。但强化的巩固相对容易些，只要间隔得给予奖励，其行为就能保持。但对于惩罚，最好的方法每次都给予及时的实施，否则会引起侥幸心理。

  操作性条件作用的应用场景很多，典型的比如有对于小孩的管教，家长可以根据实际情况选择强化或者惩罚，是正强化（获得奖励）还是负强化（消除不良刺激），是正惩罚（施加不良刺激）还是负惩罚（剥夺强化物）。另外，家长的关心也是种强化物，在可控的情况下，对孩子的不良行为不予理睬，使其行为逐渐“消退”，这也是一种方法。

认知学习

到目前为止，上述提及的心理学家都属于行为主义流派，他们坚持认为内在的心理活动属于不可观察的领域，不能作为科学研究的对象，因此他们只研究能被观察的外在行为。但上述的两个条件作用并不能完全解释人类的学习行为，有大量的实验证明了头脑中的思考对于学习的重要作用。

  首先是柯勒和他的黑猩猩，他的实验证明了黑猩猩能够通过思考把所学的技能结合起来解决问题。他的黑猩猩掌握了通过把箱子堆起来，然后爬上去摘到挂在笼子高处的水果，同时还学会了通过棍子去够到水果。接着柯勒把水果挂得更高，他的黑猩猩在多次尝试失败后，发脾气并放弃了尝试，然后它突然跳了起来，它拖着箱子和棍子来到了水果下方，站到箱子上用棍子挑获了战利品。这说明动物不是没头没脑地使用习得的行为，而是通过在头脑中的心理过程来改变他们的行为并解决问题。

  第二个实验是托尔曼和他的老鼠。他发现老鼠具有在迷宫中建立地图的能力，每当通往强化物的路径上出现障碍时，老鼠会迅速规划出另一条路径，而不是满世界乱转。即便把老鼠放在迷宫里任其自由游荡而不给任何食物奖励，老鼠仍旧会在头脑中建立起地图。这些行为无论如何也无法通过刺激和反应的简单关系来解释。

  认知学习理论认为学习是信息处理的过程，学习者从刺激中寻找有用的信息，学习方式包括顿悟，观察学习，认知地图，以及读者您现在正在进行的概念理解和记忆等复杂的形式，这些内容将在后续文章中详细展开。

相信你的老师肯定说过，活到老学到老，从行为主义的学习方式来看，这话一点没错，只要我们的大脑还在工作，它必然在不断地学习。而那些更复杂的学习，还能为我们带来满足好奇心的成就感，以及超越自身局限的智慧。
