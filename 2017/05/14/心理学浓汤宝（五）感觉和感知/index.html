心理学浓汤宝（五）感觉和感知

发表于 2017-05-14 更新于 2021-12-03 分类于 psychology

感觉是身体的感受器官感受到的刺激，感知是大脑对于感觉的解释，包括给感觉赋予的意义以及与之相关的记忆和概念。单纯的感觉只是简单的信号采集，转换成感知的能力才是人类智能的强大之处。人体的感觉很丰富：视觉，听觉，嗅觉，触觉，痛觉，平衡感觉，身体位置感觉等等，让我们选择视觉来一窥这个神奇的转换过程。

视觉源自视网膜将光能转换为神经冲动，进而传递到大脑皮层进行处理。

处理视觉的大脑相关部分看上去很复杂，而且实际上我们对其的了解可能还只是冰山一角。其中视网膜部分相对容易理解一些，如下图所示，光通过光感受器细胞，转换为神经冲动，这些神经信号经由多种神经细胞进行收集和传输。

光感受器细胞分两类，一类是只对亮度敏感的视杆细胞（约1.25亿个），另一类由三种分别对红，绿，蓝敏感的视锥细胞组成，约700万个。黑暗中，视杆细胞提供视觉信号，而明亮时，主要由视锥细胞提供。

对不同波长敏感的视锥细胞各自产生强弱不同的神经冲动，大脑综合三种神经冲动的强弱程度调合出千变万化的色彩。实际上颜色只是大脑赋予的意义，大脑给不同的神经信号赋予了特定的颜色。而对于猫，由于其强调弱光环境的视觉能力，视杆细胞数量特别多，视锥细胞较少，且只有红色和绿色两类，它的大脑无法产生蓝色的视觉，就如同人类无法产生第四种原色的感觉一样。因此颜色实际并不存在，它只是大脑对波长分类的结果。

大脑对视觉的处理远远不只是赋予颜色，它会分辨边界，区分背景和图形，提取图形特征并与搜索记忆中的匹配项，这样，当你看到一个朋友时，你会知道他是谁。大脑还会从视觉中感知深度，纠正因为视角而产生变形的图形，或者补充出被遮挡的部分，为摸棱两可的图形找到合理的解释等等。格式塔心里学家对此做了深入的研究，并归纳出一些感知定律，比如相似律和相近率，大脑会把外表相似的事物联系归在一起，或者把位置邻近的归在一起，作为整体考虑。再比如连续律，大脑会倾向于找出一条连续的线条，即便它被其他线条交织或切割。 大脑对视觉进行如此复杂的后期PS同时也带来了一个问题，那就是可能会带来错觉，在著名的赫尔曼网格中，请注意错觉在线条的交叉点上变的小魔术。

从感觉到感知的过程还会受到其他因素的影响，比如感觉的适应，当我们适应某种环境声音时，这个声音就不会再持续得出现在我们的意识里，但是当环境音量发生变化时，它会再次进入我们的意识。再比如情绪的影响，当我们处于不同的情绪时，相同的外部环境的刺激会产生不同的影响。

认识到感知并非是外部世界的真实反映，这是迈入理性思考的重要一步。
